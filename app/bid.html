eCommerce App Ecosystem

Presented by Astrexa Innovation

1. Customer eCommerce App

A full-featured mobile app for iOS and Android built with React Native. This app allows users to browse, search, and purchase products securely. Integrated with Razorpay for payments, and MSG91 for SMS notifications.

  • Product listing, filtering, and smart search
  • Wishlist, cart, and order management
  • Secure checkout and payment integration
  • Real-time order tracking and invoice download
  • SMS alerts for confirmations and delivery

Tech Used: React Native + Firebase

2. Delivery Partner App

A dedicated mobile app for delivery staff to receive, track, and complete delivery orders efficiently. It provides OTP verification, route navigation, and automatic status sync with the main system.

  • Delivery assignment and OTP confirmation
  • Live route navigation with Google Maps
  • Status updates with push & SMS notifications
  • Auto-sync with inventory system via Firebase

Tech Used: React Native + Firebase

3. Web-based Inventory & Admin System

A web-based dashboard to manage all products, inventory, users, orders, and payments. Built using Next.js and Firebase for real-time control and analytics across the platform.

  • Inventory sync with mobile apps
  • Product uploads, price, and stock management
  • User role control (admin, staff, support)
  • Live order tracking and revenue reports

Tech Used: Next.js + Firebase
